# Практична робота 8: Функції вищого порядку

## Тема

**Варіант 3: Collection Utilities**

## Мета роботи

Створити набір утилітарних функцій для роботи з колекціями даних, використовуючи функції вищого порядку, чисті функції, композицію, каррінг та методи `map`, `filter`, `reduce`.

## Реалізовані функції

У проєкті реалізовано такі функції:

1. `groupBy` — групування елементів за критерієм.
2. `sortBy` — багатокритеріальне сортування.
3. `partition` — розділення масиву на дві групи.
4. `pluck` — витягування властивостей з об'єктів.
5. `uniqBy` — отримання унікальних елементів за критерієм.
6. `pipe` — композиція функцій зліва направо.
7. `curry` — каррінг функцій.
8. `filterBy`, `mapBy`, `sumBy` — додаткові функції для демонстрації `filter`, `map`, `reduce`.

## Приклади використання

### groupBy

```js
groupBy((student) => student.group)(students);
```

Результат: студенти групуються за навчальною групою.

### sortBy

```js
sortBy([(student) => student.score, (student) => student.name])(students);
```

Результат: студенти сортуються спочатку за балом, потім за іменем.

### partition

```js
partition((student) => student.score >= 75)(students);
```

Результат: масив розділяється на студентів, які склали, і студентів, які не склали.

### pluck

```js
pluck("name")(students);
```

Результат: повертається масив імен студентів.

### uniqBy

```js
uniqBy((student) => student.name)(students);
```

Результат: повертається список студентів без повторів за іменем.

### pipe

```js
const getHighRatedStudents = pipe(
  filterBy((student) => student.score >= 80),
  sortBy([(student) => -student.score]),
  mapBy((student) => `${student.name} — ${student.score} балів`)
);
```

Результат: дані проходять через декілька функцій послідовно.
